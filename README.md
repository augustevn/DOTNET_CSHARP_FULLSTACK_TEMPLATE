# Pet

Contains multiple fullstack .NET C# Web API - Blazor WASM projects, sharing setups, business logic and styling.

Setup:
- Docker for a local MSSQL database
- Entity framework
- Identity framework
- JWT bearer authentication and authorization
- Exception handling
- DTO validation
- Mailing
- ...
